Объект-блок это область, которая может содержать текст и картинку. Вы также можете дать ваши собственные подсказки для него. Вы можете "запереть" блок, если хотите сделать его доступным только для чтения.

Типичный сценарий -- это создать несколько блоков на холсте, организовать их по желанию, "запереть" их, и затем сохранить Холст как шаблон.

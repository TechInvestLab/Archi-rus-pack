Значение (meaning) определяется как знание или опыт (содержание), присутствующие в рабочем продукте какого-то объекта с точки зрения контекста (в разных контекстах один и тот же объект может иметь разный смысл -- для кого-то это просто брошенная перчатка, а для кого-то может быть вызов на дуэль).

Значение представляет уточнение, как употреблять рабочий продукт. Оно описывает, как представление информации объекта в виде рабочего продукта будет воспринято внешним пользователем.

Определенная интерпретация представления для определенного пользователя несет определенное значение.

Имя значения предпочтительно должно быть существительным или утвердительной фразой.

Категория:

Деятельность. Информационная.

Примеры:

Объяснение (страхового) полиса, Уведомление о страховом полисе.

Типовые отношения:

Связь - с рабочим продуктом

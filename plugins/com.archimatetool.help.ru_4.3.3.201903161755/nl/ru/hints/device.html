Устройство (device) определяется как физический вычислительный ресурс, на котором информобъект может быть развёрнут для выполнения.

Устройство является специализацией "железа", представляя физический ресурс с процессорными возможностями. Устройство обычно используется для моделирования компьютерной аппаратуры, такой как мейнфреймы, персональные компьютеры или роутеры.

Обычно, устройство является составной частью "железа" вместе с системным софтом. Устройство может быть составным, то есть состоять из под-устройств.

Устройства могут быть физически связаны сетями. Устройство может быть назначено (то есть на нём могут быть развёрнуты и храниться/выполняться) на информобъекты.

Системный софт может быть назначен на устройство. "Железо" может содержать одно или несколько устройств.

Имя устройства должно быть существительным, предпочитительно ссылающимся на тип компьютерных устройств.

Категория:

"Железо". Выполнитель.

Пример:

Файловый сервер, Роутер, Мейнфрейм, Десктопный компьютер, IBM System Z.

Типичные отношения:

Состав - устройство, "железо"
Назначение - системный софт, информобъект

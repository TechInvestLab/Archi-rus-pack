Функционал связки программ (application interaction) определяется как внешне видимая единица работы, выполняемая участвующими в связке программ двумя или более программами.

Функционал связки также специфицирует внешне видимую работу, которая нужна для реализации сервиса программ.

Функционалу связки может быть назначена связка программ.

Функционал связки может реализовать сервис программ.

Функционал связки может использовать сервис программ и сервис "железа".

Функционал связки может иметь доступ к данным.

Имя функционала связки предпочтительно должно быть глаголом.

Категория:

"Софт". Работы.

Пример:

Провести транзакцию

Типовые отношения:

Назначения - для связки программ
Реализации - сервис программ
Использования - сервис программ, сервис "железа"
Доступ - к данным

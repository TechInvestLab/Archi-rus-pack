Функционал программы (application function) определяется как представление однородной группы внутренних работ программы.

Функционал программы описывает внутренние работы программы. Внутренние работы функционала невидимы для использующих программу.

Поведение функционала программы выставляется вовне через один или более сервисов программ, т.е. функционал программы может реализовать сервис программ.

Функционалом программы могут быть использованы другие функционалы программ и сервисы программ, а также сервисы "железа".

Функционал программы может иметь доступ к данным.

Функционалу программы может быть назначена программа.

Имя функционала программы предпочтительно должно быть отглагольным существительным.

Категория:

"Софт". Работы.

Примеры:

Учёт, Выставление счетов, Выписка полиса, Начисление премий, Ведение финансовых дел.

Типовые отношения:

Использование - сервисом программ
Запуск - функционала программы
Доступ - к данным
Назначение - для программной компоненты

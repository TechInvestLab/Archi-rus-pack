Метод описания использования программ (Application Usage) описывает как программы используются для поддержки одного или более процессов деятельности, и как они используются другими программами. Он может быть использован для проектирования приложений путём идентификации сервисов, нужных для процессов деятельности и других программ, или для проектирования процессов деятельности, путём описания сервисов, которые им доступны. Более того, поскольку этот метод описания идентифицирует зависимости процессов от программ, он может быть полезен для операционных менеджеров, ответственных за эти процессы деятельности.
